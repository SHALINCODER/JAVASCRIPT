# JavaScript Learning Journey

A comprehensive, structured learning resource for JavaScript fundamentals, intermediate concepts, and DOM manipulation. I have created these files for learning purposes to build a strong foundation in JavaScript. This repository contains organized lesson folders with practical examples covering everything from basic operators to advanced object-oriented programming.

### **PART - 1: JavaScript Fundamentals** (18 lessons)
Core JavaScript concepts and syntax:
- **ARITHMETIC OPERATORS** - Basic mathematical operations
- **VARIABLES** - Variable declaration and scoping basics
- **VARIABLES - 2** - Advanced variable concepts
- **CONST** - Constant declarations and behavior
- **TYPE CONVERSION** - Converting between data types
- **STRICT EQUALITY** - Equality operators (===, !==)
- **USER INPUT** - Handling user input with prompt/alert
- **FUNCTIONS** - Function declaration and invocation
- **FOR LOOP** - Iteration with for loops
- **WHILE LOOP** - Iteration with while loops
- **SWITCH** - Conditional switch statements
- **TERNARY OPERATOR** - Conditional ternary operator
- **LOGICAL OPERATOR** - AND (&&), OR (||), NOT (!)
- **STRING METHODS** - String manipulation and methods
- **STRING SLICING** - Extracting parts of strings
- **MATH** - JavaScript Math object and methods
- **METHOD CHAINING** - Chaining multiple methods
- **STARTING** - Getting started template

### **PART - 2: Advanced JavaScript** (18 lessons)
Object-oriented and functional programming:
- **ARRAY** - Array creation and manipulation
- **FOR EACH METHOD** - Array iteration with forEach()
- **MAP** - Transforming arrays with map()
- **FILTER METHOD** - Filtering arrays
- **REDUCE METHOD** - Reducing arrays to single values
- **FUNCTION EXPRESSION** - Functions as expressions
- **ARROW FUNCTIONS** - ES6 arrow function syntax
- **CALLBACK FUNCTION** - Functions as arguments
- **REST PARAMETERS** - Rest parameters in functions
- **SPREAD OPERATOR** - Spread syntax for arrays/objects
- **OBJECTS** - Object creation and manipulation
- **CONSTRUCTOR** - Constructor functions
- **CLASS** - ES6 class syntax
- **THIS** - Context and this keyword
- **GETTERS AND SETTERS** - Property accessors
- **STATIC** - Static methods and properties
- **INHERITANCE** - Class inheritance with extends
- **SUPER** - Super keyword for parent methods
- **VARIABLE SCOPE** - Function and block scope

### **PART - 3: DOM & Advanced Concepts** (12 lessons)
DOM manipulation and advanced features:
- **DOM** - Document Object Model basics
- **ELEMENT SELECTORS** - querySelector, getElementById, etc.
- **DOM NAVIGATION** - Traversing DOM tree
- **ADD AND CHANGE HTML** - Modifying HTML content
- **ARRAY OF OBJECTS** - Working with complex data structures
- **NESTED OBJECT** - Objects containing objects
- **DESTRUCTURING** - Destructuring assignment
- **CLOSURE** - Understanding closures
- **DATE OBJECT** - Working with dates
- **SETTIMEOUT FUNCTION** - Timing functions
- **SHUFFLE AN ARRAY** - Array randomization
- **SORTING** - Sorting arrays
- **SYNC VS ASYNC** - Understanding synchronous vs asynchronous code
- **ERROR OBJECT** - Error handling
- **ES6 MODULES** - Module import/export

Practical applications combining multiple concepts:
- **KEY EVENTS** - Keyboard event handling
- **MOUSE EVENTS** - Mouse event handling
- **CLASSLIST** - Dynamic class manipulation
- **HIDE-SHOW HTML** - Toggling element visibility
- **NODELIST** - Working with NodeList

## üöÄ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/javascript-learning.git
   cd javascript-learning
   ```

2. **Open a lesson folder** - Each folder contains:
   - `index.html` - HTML structure
   - `index.js` - JavaScript code
   - `style.css` - Styling (where applicable)

3. **Run in browser** - Open `index.html` in your web browser to see the examples in action

4. **Learn and experiment** - Read through the JavaScript code, modify it, and test your changes in the browser console
